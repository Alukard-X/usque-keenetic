```markdown
# Usque Auto-Installer for Keenetic

Автоматический установщик и конфигуратор [Usque](https://github.com/Diniboy1123/usque) для роутеров Keenetic с установленным Entware.

Скрипт полностью автоматизирует процесс: от определения архитектуры процессора и скачивания актуальной версии бинарника, до настройки автозапуска и регистрации устройства.

## Возможности

*   **Автоопределение архитектуры:** Корректно определяет `mipsle`, `armv7`, `arm64` и другие архитектуры, используемые в роутерах Keenetic (включая определение Little Endian для моделей на MIPS).
*   **Умная установка зависимостей:** Проверяет наличие `wget-ssl`, `ca-certificates` и `unzip`, и устанавливает их только при необходимости.
*   **Скачивание последней версии:** Получает ссылку на последний релиз напрямую с GitHub API.
*   **Автоматическая конфигурация:**
    *   Определяет внутренний IP-адрес роутера для привязки сервиса.
    *   Создает скрипт инициализации `init.d` для автозапуска.
*   **Регистрация:** Автоматически принимает лицензионное соглашение и регистрирует `usque`.

## Требования

*   Роутер Keenetic с поддержкой Entware.
*   Доступ к роутеру по SSH.
*   Подключенное хранилище (USB-накопитель) для установки Entware.

## Установка

Подключитесь к роутеру по SSH и выполните следующую команду:

opkg install wget-ssl ca-certificates

wget -qO- https://raw.githubusercontent.com/Alukard-X/usque-keenetic/refs/heads/main/install_usque.sh | sh

```

*Примечание: Если вы еще не установили Entware, скрипт не сработает. Сначала установите Entware.*

## Использование

После завершения установки сервис `Usque` будет автоматически запущен и добавлен в автозагрузку.

### Управление сервисом

Для управления используется стандартный скрипт в `/opt/etc/init.d/`.

*   **Статус:**
    ```bash
    /opt/etc/init.d/S99usque status
    ```
*   **Остановка:**
    ```bash
    /opt/etc/init.d/S99usque stop
    ```
*   **Запуск:**
    ```bash
    /opt/etc/init.d/S99usque start
    ```
*   **Перезапуск:**
    ```bash
    /opt/etc/init.d/S99usque restart
    ```

### Настройка параметров

Если вам нужно изменить параметры запуска (например, порт, DNS сервера или привязку к другому IP), отредактируйте файл конфигурации:

```bash
nano /opt/etc/init.d/S99usque
```

Найдите строку `ARGS` и измените параметры по вашему усмотрению:

```sh
ARGS="socks -S -b 192.168.1.1 -p 8480 -d 1.1.1.1 -d 1.0.0.1 -s ozon.ru"
```

После редактирования перезапустите сервис:
```bash
/opt/etc/init.d/S99usque restart
```

## Как это работает

1.  Скрипт проверяет наличие необходимых утилит (`unzip`, `wget-ssl`, `ca-certificates`) через `opkg`.
2.  Определяет архитектуру процессора, используя базу данных пакетов Entware (`opkg print-architecture`), что исключает ошибки определения `mips` vs `mipsle`.
3.  Запрашивает GitHub API для поиска ссылки на последний релиз, соответствующий архитектуре роутера.
4.  Скачивает, распаковывает и перемещает бинарный файл в `/opt/usr/bin/usque`.
5.  Определяет IP-адрес LAN-интерфейса (обычно `br0`) для параметра `-b`.
6.  Создает скрипт запуска `S99usque`, использующий `start-stop-daemon` для корректной работы в фоне.
7.  Выполняет команду `usque register` и запускает сервис.

## Поддерживаемые архитектуры

*   `mipsle` (Little Endian) — большинство моделей Keenetic (Giga, Ultra, Extra, Viva и др. на MIPS).
*   `mips` (Big Endian) — редкие модели.
*   `armv7` (32-bit ARM).
*   `arm64` / `aarch64` (64-bit ARM).
*   `x86_64` (x86).
